# Vet Farias - Sistema de Gestão Interna

Sistema web de gestão interna para a clínica veterinária **Vet Farias**.

## Funcionalidades

- 🔐 **Login/Autenticação** - Acesso restrito com credenciais
- 🧪 **Coletas Laboratoriais** - Controle de custos de exames terceirizados com upload de Excel
- ✂️ **Volantes Cirurgiões** - Registro de procedimentos de cirurgiões terceirizados
- 📷 **Volantes Exames de Imagem** - Controle de exames de imagem (USG, RX, ECG, ECO)
- 💰 **Gastos Diversos** - Registro de gastos do dia a dia
- 💳 **Maquinetas** - Controle de recebimentos com alertas de limite
- 📊 **Relatórios Mensais** - Gráficos e análises para cada módulo

## Tecnologias

- Next.js 16
- Chart.js
- SheetJS (xlsx)
- React Icons

## Como Executar

```bash
npm install
npm run dev
```
